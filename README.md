# 📚 API Java

API REST desenvolvida com **Java** e **Spring Boot**, implementando operações CRUD completas para gerenciamento de recursos. O projeto segue boas práticas de desenvolvimento, incluindo arquitetura em camadas, tratamento de exceções e testes unitários.

## 🚀 Tecnologias

- Java 25
- Spring Boot
- Spring Web
- Spring Data JPA
- Maven
- JUnit 5
- Mockito
- H2 Database (para testes)
- PostgreSQL/MySQL (opcional)

## ✅ Funcionalidades

- Criar registro
- Buscar todos os registros
- Buscar registro por ID
- Atualizar registro
- Excluir registro
- Validação de dados
- Tratamento global de exceções
- Testes unitários da camada de serviço

## 🧪 Testes

Os testes foram desenvolvidos utilizando:

- **JUnit 5** para criação dos testes.
- **Mockito** para simulação (mock) das dependências.
- Testes cobrindo cenários de sucesso e exceções.

## 🎯 Objetivo

Este projeto foi desenvolvido com foco em aprendizado e prática de desenvolvimento backend utilizando Spring Boot, além da aplicação de testes unitários para garantir a qualidade e confiabilidade do código.
